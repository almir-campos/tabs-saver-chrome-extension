  • Hints
    • Click on gear icon near tabs group name to get group control menu, where:

    • Cross icon - deleting group

      Plus icon - add selected tab in window to the group

      Disk icon - update group with the current opened tabs in window

      Select cursor icon - rename group

    • Use ctrl+click or middle mouse button if you do not need close already opened tabs and open one of saved tabs group.
    • Use shift+click if you wanna open tabs group in new window when default mode is used. Use shift+click if you wanna open tabs group in current window when "new window" mode is used.
    • You may use "enter" key when you rename, or save tabs group.
